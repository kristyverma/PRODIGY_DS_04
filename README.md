# PRODIGY_DS_04
Overview
This project leverages Power BI to analyze customer opinions and sentiment trends from social media data. By examining tweets related to major airlines, the dashboard provides insights into customer sentiment distribution, reasons for dissatisfaction, and airline-specific performance.

🔍 Key Features
Sentiment Distribution

Total Tweets Analyzed: 15,000+

Positive: 2,363 (~16%)

Neutral: 3,099 (~21%)

Negative: 9,178 (~63%)

Airline vs Sentiment

Comparative breakdown of sentiment across airlines (American, Delta, Southwest, United, US Airways, Virgin America).

Negative sentiment dominates across all carriers, with United and American among the most discussed.

Negative Reasons Analysis

Top issues include:

Customer Service Problems

Late Flights

Cancelled Flights

Lost Luggage

Most Discussed Airlines

United, American, Delta, US Airways, Southwest, Virgin America.

📈 Visualizations
The dashboard includes:

Sentiment Distribution Chart (positive, neutral, negative)

Airline vs Sentiment Comparison

Negative Reasons Breakdown

Most Discussed Airlines Ranking

🛠️ Tech Stack
Tool: Power BI

Data Source: Twitter API (sample dataset of 15K tweets)

Techniques: Sentiment classification, categorical encoding, interactive dashboards

🚀 Insights & Applications
Identify key drivers of negative sentiment for airlines.

Benchmark airline performance based on customer opinions.

Provide actionable intelligence for improving customer service and operational efficiency.

Enable marketing teams to monitor brand reputation in real time.
